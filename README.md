# Example App

Example of Angular development using JSPM.


## Requirements

- git;
- nodejs v4+ with npm v3 (npm v2 probably work).


## Setup

You simply clone the app and install the dependencies:
```
git clone https://github.com/singpath/example-app.git
cd example-app
npm install
npm start
```
